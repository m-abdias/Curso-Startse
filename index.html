Sobre Mim

Conheça um pouco da minha história

Venho de uma carreira profissional há mais de 7 anos de experiência nas áreas de finanças das empresas, responsável por otimizar processos e torná-los autossuficientes pelas empresas a qual eu passei. Sou formada em Engenharia de Produção, apaixonada por lógica e sempre em busca de conhecimento. Meu propósito é impactar positivamente a vida das pessoas, por meio de processos mais fluídos, eficazes e inteligentes. Estou em processo de transição de carreira para a área de tecnologia. Iniciei em 2022 minha jornada na programação com formação FullStack de desenvolvimento.

Meus Projetos

Projeto Landing-Page

F4Bank

O F4Bank é a criação de um projeto de um banco público brasileiro para competir com as startups que tem surgido, como Nubank, Banco Inter e Banco Neon. Esse projeto foi desenvolvido em grupo, pensamos, desenhamos e desenvolvemos um site moderno e responsivo.

Linguagens utilizadas: HTML, CSS, JS

Projeto WhatsLab

Aplicativo de troca de mensagens

O WhatsLab foi um projeto desenvolvido por mim, que consiste em criar um aplicativo de troca de mensagens. O MVP deste projeto era: cada mensagem deve possuir um remetente (ou seja, um usuário que enviou) e o conteúdo em si. Os inputs devem ser controlados e os campos resetados após o envio da mensagem.

Linguagem utilizada: React.js

Projeto LabeCommerce

E-commerce de camisetas com estampas divertidas relacionadas ao espaço

Este projeto foi desenvolvido em grupo, com o objetivo de desenvolver um e-commerce simples utilizando o React. O site não esta responsivo, Mas consta uma home onde renderiza todos os produtos, ordenando-os com filtros. É possível adicionar os produtos ao carrinho e remover ítens do carrinho.

Linguagem utilizada: React.js

Projeto Labefy

Site para escutar suas músicas

O Labefy foi um projeto desenvolvido por mim, a ideia principal deste projeto foi realizar uma integração com a API Labefy, que permite o gerenciamento básico de playlists de músicas. O usuário é capaz de criar suas playlists, adicionar músicas as playlists criadas, escutá-las e também é possível deletar músicas e playlist.

Linguagem utilizada: React.js
